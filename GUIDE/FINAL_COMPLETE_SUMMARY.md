## 🧠 KEY FEATURES EXPLAINED

### Emotionally Intelligent AI
**What It Does:**
- Detects user's emotion: overwhelmed, anxious, sad, lonely, stressed, confused, hopeful, calm, angry, uncertain
- Measures emotion intensity (0-1 scale)
- Generates context-specific system prompts
- Adapts responses to emotion state
- Provides personality-customized support

**How It Works:**
- User types message
- EmotionDetector analyzes keywords + intensity
- SystemPromptGenerator creates adaptive prompt
- LLMService generates response (mock or real Llama)
- Response streamed to UI with typing animation
- Conversation history tracked for continuity

### Crisis Safety System
**What It Does:**
- Detects 15+ crisis phrases (suicide, self-harm, etc.)
- Shows helpline resources immediately
- Non-judgmental, compassionate intervention
- Indian crisis resources (iCall, Vandrevala, AASRA)

### Personality Customization
**Available Modes:**
- **Gentle:** Soft, validating, listening-focused
- **Curious:** Questions, exploratory, thinking-deepening
- **Wise:** Reflective, perspective-giving, thoughtful
- **Balanced:** Mix of all three (default)

User selects during onboarding, can change in settings.

### Local AI Model
**Current (Phase 1):** Mock responses that feel real
**Future (Phase 2):** Llama 3.2 1B running on-device
**Benefits:**
- No internet needed after setup
- Complete privacy (data never leaves phone)
- Works offline
- Free forever

## ⚙️ TECHNICAL ARCHITECTURE

### Stack (All Free & Open Source)
- **Framework:** Flutter 3.x
- **State:** Riverpod 2.x
- **Database:** Isar (encrypted local)
- **AI:** flutter_gemma (Llama 3.2 1B)
- **UI:** Material Design 3 + Cupertino
- **Fonts:** Lora (serif) + DM Sans (sans)

### File Organization
```
lib/
├── main.dart
├── config/              # Constants, AI config
├── theme/               # Colors, typography
├── models/              # Data classes
├── services/            # Emotion, LLM, Crisis, Storage
├── providers/           # Riverpod state
├── screens/             # 7 screens
├── widgets/             # Reusable components
└── utils/               # Extensions, helpers
```

### Data Flow
```
User Message
    ↓
EmotionDetector (what are they feeling?)
    ↓
CrisisDetector (are they safe?)
    ↓
SystemPromptGenerator (create emotional prompt)
    ↓
LLMService (get response)
    ↓
Chat UI (show response with animation)
    ↓
Isar DB (save message)
```
